Overview
========

wneq is a python package for computing nuclear equilibria from
`webnucleo <http://webnucleo.org/>`_ files.

Installation
------------

Install from `PyPI <https://pypi.org/project/wneq>`_ with pip by
typing in your favorite terminal::

    $ pip install wneq
